Web Calculator

This is a simple, web-based calculator application which can be used to perform basic arithmetic. Either a mouse or a keyboard can be used for input. I hope you find it useful!

This application also has a unique feature: calculation results can be edited without re-entering them manually. Most calculator applications immediately clear the output when entering a new expression, but this one requires the user to clear the viewport manually if this is desired. This behaviour may be useful when composing complex expressions from simple ones.
